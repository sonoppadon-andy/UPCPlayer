UPC 2026 · PLAYER DNA WEB APP
=================================

ไฟล์ในชุด
---------
1. index.html
   เว็บแอปแบบไฟล์เดียว เปิดใช้งานได้ทันทีและไม่ต้องติดตั้งไลบรารีเพิ่มเติม

วิธีทดลองบนเครื่อง
-----------------
- ดับเบิลคลิกไฟล์ index.html เพื่อเปิดด้วย Chrome, Edge, Safari หรือ Firefox
- แบบประเมินจะคำนวณผลในเบราว์เซอร์ของผู้ตอบ
- ผู้ตอบสามารถคัดลอกผล ดาวน์โหลดไฟล์สรุป หรือพิมพ์/บันทึกเป็น PDF ได้

วิธีนำขึ้นใช้งานจริง
--------------------
- เปลี่ยนชื่อไฟล์เป็น index.html (ภายในชุดนี้เปลี่ยนไว้แล้ว)
- อัปโหลดไปยัง Static Web Hosting, Intranet, SharePoint ที่รองรับหน้า HTML,
  GitHub Pages, Netlify, Vercel หรือเว็บเซิร์ฟเวอร์ขององค์กร
- ทดสอบ URL จริงบนมือถือและคอมพิวเตอร์ก่อนส่งให้พนักงาน

การเก็บข้อมูล
-------------
ค่าเริ่มต้นของเว็บแอปจะไม่ส่งคำตอบออกจากอุปกรณ์ และไม่มีฐานข้อมูลส่วนกลาง

หากต้องการเก็บผลลัพธ์รวม:
1. เปิดไฟล์ index.html ด้วยโปรแกรมแก้ไขข้อความ
2. ค้นหา APP_CONFIG
3. ใส่ URL ของ Webhook/API ในค่า submissionEndpoint
4. Endpoint จะได้รับข้อมูล JSON ประกอบด้วยชื่อเล่น ทีม/BU คำตอบ คะแนน
   บทบาทหลัก บทบาทรอง และเวลาที่ทำแบบประเมินเสร็จ

ตัวอย่าง:
submissionEndpoint: "https://your-company.example/api/player-dna"

ข้อแนะนำด้านข้อมูลส่วนบุคคล
---------------------------
- ชื่อเล่นและทีม/BU เป็นข้อมูลไม่บังคับ
- หากเปิดระบบเก็บผลส่วนกลาง ควรแจ้งวัตถุประสงค์ ระยะเวลาจัดเก็บ
  ผู้มีสิทธิ์เข้าถึง และช่องทางติดต่อผู้ดูแลข้อมูลให้ผู้ตอบทราบ
- ผลลัพธ์นี้ออกแบบเพื่อกิจกรรมเรียนรู้และการสร้างทีม
  ไม่ควรใช้เป็นเครื่องมือประเมินผลงานหรือการตัดสินใจด้านบุคลากรเพียงลำพัง

เบราว์เซอร์ที่แนะนำ
-------------------
Chrome, Microsoft Edge, Safari และ Firefox เวอร์ชันปัจจุบัน
